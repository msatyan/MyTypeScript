# TypeScript Sample

Sample TypeScript with VS Code development env. 

```bash
npm install
``` 

Run "tsc" to compile the TypeScript code  
From VS Code: <ctrl>+Shift+B 
tsc -w to run on watch mode
```bash
tsc  
```

npm start  (to run the development server (lite-server))
```bash
npm start
```
